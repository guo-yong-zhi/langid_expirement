Jean Baptiste Lamarck  d aserdas ilan  d  Afṛensaw n teɣlent, akken iga daɣen d amesnudran,  d akadim d amsalel seg tazwara i tekti n teẓri n temhazt tasnudrant d akken-it  anelfu  i d-illan yerna ikemmel ilmend n yisuḍaf igamanen
Lamarck ittekka deg umsenɣi n 7 n yiseggasen mgal Prusya,  yewwi-d tazalɣi n tebɣest-is deg wannar n umennuɣ,  sin akkin igguǧ ɣer Munaku,  yuɣal Lamarck yefka azal i umezruy n ugama akken  iy guccel ɣef tezrawt n tesnujjya, yestaɣ seg tsardast ass mi yeffudi deg useggas n 1766 i wakken ad yuɣal ɣer tezrawt  n tesnujjya 
Lamarck yessnerni azal i tesnimɣant seld imi i d-yezzuzer amahil igebren seg 3 n yimaddaden ) , yewwi-d tigmemt takadimt tafransist i tmussniwin deg useggas n 1779, Lamarck yuɣal d aselmad n tesnɣersiwt ass mi i d-tesbedd tiddukla taɣelnawt tafransist asalay aɣelnaw i umezruy agaman deg useggas n 1793
Deg useggas 1801 yezzuzer-d adlis "anagraw n tnaɣlalin" igan d amahil-is agejdan deg usesmel n tnaɣlalin , d irem yessaleɣ-it deg useggas 1802
Lamarck yettumeggez d amezwaru isxedmen irem n tesnudert s unamek-is atrar, ikemmel amahil-is am umnir agejdan deg tesnɣersiwt n tnaɣlalin.
Lamarck yettumektayed deg tallit tatrart s teẓri n tukkest n tmeskanin inelmiẓen i yettusemman tukkest taleggaɣt, dɣa ibder-itt-id deg udlis-is afelsaf n tesnɣersiwt deg useggas 1809, maca tikti n teẓri-ayi tella send n usnefli n temhazt ayyi yerna tessileɣ-d anagar kra n wamur meẓẓiyen seg teẓri n temhezt, akken i tella tettwaddeg  deg tallit-is sɣur kra n yimazrayen igamanen. 
uttekki n Lamarck yettumeggez deg  temhazt d tiẓri tamezwarut i yettwaṭṭfen i temhezt tasnudrant  anda idemmer tadusi n tseggugra takrurant imuddiren ɣer teskala n usukres,  tguwes tadusi n twennaṭ s ussezgi-ines d twennaṭin tidiganin s usexdem-ines iferdisen,  d ukkas wid nniḍen,  d usemgired gar-asen,  d yimuddiren niden
