Rugrat d yiwen n unagraw n tiliẓri imarikaniyen i d-ixelqen Arlene Klasky, G ábor Csup ó, d Paul Germain i Nickelodeon. Taslit tessers iman-is ɣef yiwet n tegrawt n warrac. Imezdaɣ ameqqran : Tommy, Chuckie, Angelica, akked akniwen Phil d Lil, akked tudert-nsen yal ass, yennum țekkan tarmit n tudert i yuɣalen d tameddurt tameqqrant tameqqrant deg tisuginin n yiɛeqliyen imeqqranen.
Amaslay-nni yefka-d amezwaru deg 11 Tubeṛ 1991, d Nicktoon wis sin—deffir Doug akked deffir The Ren & Stimpy Show, d amezwaru n 65 n yixeddamen nnig tlata n yiseggasen. Tazzelt-nni teḥbes deg 1993, s tazzelt taneggarut n usafag deg 12 Novembr 1994. Deg 1995 d 1996, sin n temsirin n tmurt n Yahuda i d-yefkan azal. "A Rugrats Passover" d "A Rugrats Chanukah", i sin yid-sen yewwi-d asfel d ameqran. Deg lweqt-a, mi ɛeddan tazzla n umezgun, Rugrat yebda yettnerni s lewɛadi d lɛaḍima ɣef ddemma n uɣallen n temlilit deg Nickelodeon.
