José Rizal   yella d amussnaw d aɣelnaw  n Filipin. D netta d amdan ameqqran i d-yessekren asudu n Filipin deg tallit n anekcum aspenyul.  Yettwaḥseb d agrawli aɣelnaw n Filipin, ass n tmettant-is yettusmekti-d d yiwen n wass aɣelnaw yettusemman Ussan n Rizal. José yella d aqcic wis-sin deg yiwet n twacult d taẓayant izedɣen deg kalamba deg twilayt n Laguna, anda i d-yewwi agerdas deg tẓuri. José yezweg ɣer Madrid anda ikemmel leqraya-s deg tesdawit n Kamplotensi yewwi-d dinna agerdas n ujijji. Yeṭṭef amḍiq ɣer Tasdawit n Paris  fr , syin yewwi-d aswir wis sin n tdoctura deg tesdawit n Heidelberg. Rizal yella yessawal mraw n tutlayin,  yella ula d amedyaz, d amaru,  yura aṭas n wungalen.
