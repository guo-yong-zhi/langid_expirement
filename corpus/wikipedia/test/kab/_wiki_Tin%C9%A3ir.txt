Tinɣir  d tamdint id yezgan deg tama n usamar n Tgelda tameṛṛukit deg uzaɣar n Tudɣa ar tuffɣa n Weṭlas Afellay af udem n Wedrar n Saɣru. D tamaneɣt n Tasga n Tiɣir seg useggwas n 2009. Deg 2004 akk iɣerman d wemdan n tɣiwant-agi ɣur-sen azal n  86.500 imezdaɣen tuget-nsen d imaziɣen
Akk imezdaɣen llan d imaziɣen, ttmeslayen akk gar-asen s Tamaziɣt n weṭlas alemmas maca qqaren tutlayt nnsen tacelḥit. Deg temnaṭ, llant ttzdiɣent tiwsitin timaziɣin am Ayt Ɛeṭṭa ar tama n wenẓul, Ayt Merɣad deg wegmuḍ, Ayt Ḥdidu deg ugafa et Ayt Yafelman.
