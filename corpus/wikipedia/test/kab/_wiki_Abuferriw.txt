Abuferriw neɣ Tawacult n ibuferriwen  d tawacult n yigṭaṭ yettidiren deg umadal amzikan d Marikan n Ugafa, Abuferriw d afrux meẓẓiyen, ableɣ-is yettili gar n 10 cm ar 22 cm, Aṭas seg telmas n twacult-a ttidiren zdat n yimdanen anda ttafen učči-nsen am yifsan.
Anagar tiwsitin tiynelmasin, yakk tilmas n ubuferriw d tid yettemcabin s waṭas deg talɣa. Dɣa tawacult-a tettwassnen s wemcabi n yigmamen-is  akk. Teɣzi tagejdant n telmas n ubuferriw yettili gar n 10 alamma 16 cm, yakk d 9 alamma d 21 cm i tewsitin tiynelmasin, ma d azuk yettili gar n 7 ar 29 g yakk d 5 ar 49 g i tewsitin tiynelmasin, S umata ifurken igejdanen n ubuferriw ttaken-d seg yiniten yemgaraden n wafrawen-is. 
Aqamum 
Aqamum n ubuferriw meẓẓiy maca llan wid ihrawen d wid sdiden  s unect  n wučči ɣef wacu yegguni, Acku tilmas iy tetten iburɣas lant  iqamumen sdiden, ma d tilmas iy tetten ifsan iqamumen-nsent zhur-it.
Abuferriw d ajqiw yettidiren deg waṭas n yidgan n umadal anda yettidir deg umur ameqran n imenẓawen n Uruppa d Asya d Tefriqt d Marikan n Ugafa, Ula di tmurt n Iqbayliyen llant kra n telmas am ubuferriw azegzaw.
S unect n usismel n umnir  n uswir amesnaylal agreɣlan
