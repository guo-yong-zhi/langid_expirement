Pencarian Windows  adalah aplikasi pencarian berkas yang terdapat pada Sistem Operasi Microsoft. Aplikasi ini dikenalkan sejak Windows Vista sebagai pengganti pencarian pada Windows 2000 dan MSN Desktop Search pada Windows XP juga Windows server 2003. Mesin pencarian ini dikembangkan setelah WinFS dan keberlanjutan operasi sistem windows sebagai kelebihan dari platform.
Pencarian Windows ini sebagai kelanjutan dari kesuksesan Indexing Service. Microsoft memulai mengembangkan mesin ini setelah WinFS yang mengenalkan fitur original dari kelebihan platform seperti pengindeksan konten, pencarian bertahap dan penumpukan properti.
