Jalur kereta api Glen Waverley adalah jalur kereta api komuter Public Transport Victoria  di Melbourne, Victoria. Jalur ini bercabang dari Jalur Lilydale, Belgrave, dan Alamein di Burnley, dan melintasi Sungai Yarra untuk menuju Monash Freeway dan lembah Gardiners Creek melalui tujuh stasiun menuju ke Holmesglen. Selanjutnya jalur ini menuju stasiun Jordanville dan Mount Waverley, sebelum mencapai puncak ketinggian 130 meter  di atas permukaan laut di sebelah barat Jalan Lawrence. Setelah ini, jalur turun melintasi lembah sebelum stasiun berikutnya, Syndal. Jalur ini naik dan turun lagi hingga menuju ke terminus, Glen Waverley.
