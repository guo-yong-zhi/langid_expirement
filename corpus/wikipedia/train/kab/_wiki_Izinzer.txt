Izinzer  d tafesna n ibureɣsen yeṭṭafaren afelfesna n tgensefriwin deg wadusmil n temsefriwin 1  2  3 .Tilmas-is yettwassnen ugtent s waṭas, anda ttawṭent ɣer 400.000 n telmas neɣ ayen yegdan 40% n yiburɣas i d-ittwasnemlen, ayen i d-immalen d akken iznezren i d ibureɣsen yakk ɣef wacu i d-yella usenmel d weglam ugar n ibureɣsen niden, anda i d-ssiliɣen azal n 30% n tudert yettwassnen acku llant wazal n 5 alamma d 8 n imelyan n telmas yettwassnen d tid Ur nettwassen.
Dɣa, ar tura mazal yettili-d usnirem d tifin n telmas timaynutin war tamuɣli ɣer telmas inegren neɣ tid yuɣalen d azṛu.
Izinezren ttidiren deg wamur ameqran n twennaṭin, ama deg yigran, deg tẓegwa neɣ deg tneẓrufin maca ur ttidiren ara deg yideggan anda yettili usemmiṭ am Antraktika. 
Izinzer itett imɣan d traɣlalin d wadalen, akken iy tteɣẓaẓ isekla imidduren d wid yeɣlin s agensu n tadriwin-nsent. Akken i llant kra seg telmas-is d tigemrawin tetten ibureɣsen niden neɣ deg tikwal niden tetten ula d yifrax n yigṭaṭ.
Izinezren ttemyarawen s temhazt tummidt* anda tawtemt tessrusay seg kra n imṛawen, n twinas alama d igimen n tmellalin deg teɣzi n tudert-is, deg yal tikkelt tessrusay timellalin-is deg wadeg imezgi anda yettili wučči i tkeččawin-is mi ara ad frurxent. Yerna tikeččawin n izinzer mgaradent ɣef tkeččawin n ibureɣsen niden deg tmeẓrit-is taqurant d yixef-is amsulles akken i llan yiriwen deg yimi i twaffẓa d yimrujen n wesgunef deg tɣzi n tfekka-ines.
Deg talit n usiken, izinezren sskanayen-d tikeli tukrist, anda tikdi  tga d amur axatar deg temhelt n usiken. Ma yella ɣer kra n telmas n twacult n nicrophorus yettili-d umennuɣ gar n twetmin anda tid-ak yettwarnan  tteffɣent seg wannar n usiken s umṭafar anda i d-ttegrayen anagar izinezren ussisen  i usegger n usiken yakk d izenzer ussis.
Deg tuget n tegnatin tettili-d tallit n usiken d tin meẓẓiyen anda tettawi kra b tsinin neɣ tesdatin. Maca ɣer kra n telmas tettkemmil temhalt n wuggi aṭas n tesragin anda i d-tettili tuddit n uẓẓel seg iwetmen ɣdr twetmin sin akin tteylullusent tmellalin i wakken ad ttwasersent, dɣa asrusu n tmellalin yemgarad seg telmest ɣer telmest niden, acku kra n telmas ssrusayent timellalin-nsent deg yiferran n yimɣan ma d tilmas niden zeṭṭen-tt-d tadliwin n weɣris deg yiferran n yimɣan sin akin ad ssersent deg-sent timellalin i wakken ad ttwaggẓent akken iy ssefk. Akken daɣen i llant telmas yeɣɣazen igedfen deg umezzur neɣ zdat am uẓerri iggaren timellalin-is s agensu  n yiẓẓan sin akin ad ten-yerr d tuzza  w ad tizzuɣer s yiṭarren-is n deffir ɣer wangu-ibes .
