Imenɣi amezzwaru gar Legliz d Afɣanistan yebda deg 1839 yekfa deg 1984 deg-s teɛreḍ tgelda yedduklen ad tesseɣli Afɣanistan deg tallit n wurar ameqqran id-yellan gar-as akked Rrus ɣef umenkud ara iseeɣlin amur ameqqran n tmura deg Asya. Tfukk s rrbeḥ n Afɣanistan.
Yextar-d udabu n Lhend yiwen n uɣalla-ines qqaren-as Alexander Burnes, i yettmesslayen tutlayin n Urdu d Dari akken ilaq lḥal  yernu yessen aṭas seg tutlayt tabactunit  akken ad iwali liḥala n tsertit deg Kabul. Tedda yid-s terbaɛt meẓẓiyen, yeqqim dinna acḥal n tikal seg 1836 ar 1838. Iserdasen yusa-ten-id Lamer i wakken ad kecmen ɣer Afɣanistan  macci akken yella yebɣa Burnes, i wakken ad ɣeḍlen agellid n Afɣanistan n yimiren Dust Muḥemmed , i yellan d aɛdaw-nsen sersen deg wadeg-is ageldun aqbur i wumi yettwakkes leḥkem Cah Cuǧa. Kecmen yebriṭaniyen Kabul deg ɣuct n 1839. Cah Cuǧa yuɣal yettnusu deg teɣremt yellan deg leswar n Bala Ḥisaar.
Liḥala ur teɛǧib ara agdud aɣelnaw ur neḥmil ara inembaḍen iberraniyen. Tekkezr tegrawal ɣef ufus n Wazir Akbar Xan, mmi-s n Dust Muḥemmed. Yettwanɣa Burnes i yuɣalen d immetel n umenkud abriṭani deg wass n 2 unbir 1841 akked yiɛeggalen n terbaɛt-is. Fɣen yinglizen seg Kabul deg yennayer 1842. Yerwel uzal n 16 500 n yimdanen  ɣer Jalalabad . Maca akken ma llan ttwanɣan akk neɣ ttwaṭṭfen deg tnegluft n Gandamak imi imdanen mazal tassa-nsen terɣa ɣef ucengu-agi. Yiwen kan id-yuɣalen d amuddir d amejjay Wiliam Brydon. Ma d Cah Cuǧa yettwanɣa deg 5 yebrir 1842.
Igliziyen llan bɣan ad rren mliḥ ttaṛ. Snat n trebbaɛ tibriṭaniyin kecment Afɣanistan seg waṭas n tnillatin deg unebdu n 1842. Ṭṭfen Kabul deg ctumber. Ufan-d kra seg widak yettwaṭṭfen deg yennayer. Maca imi mazal reffu deg wulawen-nsen yefka uɣella Pollock lamer iwakken ad serɣen agadez n Kabul maca bdan yigduden iɣelnawen tigrawliwin dɣa imir imir fɣen-d seg Afɣanistan yuɣal-d Dust Muḥemmed ɣer unbaḍ armi d lmut-is deg 1863.
Terwi liḥala mi bdan warraw n Duyst Muḥemmed amennuɣ ɣef leḥkem yekker nnfdaq n daxe'l irusiyen zeḍmen-d seg ugafa ma d ifarisiyen seg umazlu ibriṭaniyen llan ḥtaren ma d kecmen aḥric n usammer n Faɣanistan neɣ ad ɛiwnen tuffɣa n tmurt tilellit. Di taggara anembaḍ n Kabul Yeɛqub Xan i iɣeḍlen baba-s Cir Ɛli mmi-s n Muḥemmed Dust azref n uceyyeɛ n terbaɛt tadiblumasit ɣer Kabul i wakken ad ili talwxit akked yirusiyen yessewsaɛen akter d wakter. Ferḍn yinglizen fella-s lɛehd n Gandumak deg 26 mayu 1879 i yettaken i yibriṭaniyen azref n leqdic deg tsertit n Afɣanoistan d tmura toiberraniyin ma d lamir ad yesɛu lǧehd s daxel n tmura yella deg tlisa ara yettwasunnɣen iwayen ara yuɣalen tura d Afɣanistan.
