Tasekkurt  d assaɣ imezdi id yemmalen kra n telmas n iylalen iy lan  tiseknin timarwaswasin yakk yemmudsen  igan d tagemrawt, Tisekrin d tiyuzaṭ s tiddi meẓẓiyen. Tizrawin tussnanin ssknen-t-d d akken irem n Tsekkurt ur yesmutur ara yakk tilmas id yekkan seg yiwen n umraw. S yirem n iden tagrawt n tsekrin tga d azunazɣer  ayen yennumken  d akken tasekkurt mači d tamsartut. Dɣa tisurta n tesnudert am win n IOC yessismil amur ameqran n iylalen-a deg tɣult wadu-twacult n tsekrawin
Tasekkurt d aylal yeẓlin  s waṭas deg wafraw-is d uferfer-is, llant waṭas n telmas n tsekrin deg umadal. Talmest igan d timezdit deg tmurt n Iqbayliyen d tasekkurt tamaziɣt, ma deg Uruppa d tasekkurt tazunɣidt yakk d tasekkurt tazeggaɣt yellan s tuget 
TTiddi n tsekkrin yeṭṭafaren adu-twacult n tsekrawin d win igan gar n tjira meqqren d yifred meẓẓiyen, Učči n tsekrin yezzi ɣef yifsan id tt-affen-t deg tegnit
