Xartum d tamaneɣt n Sudan ,Zedɣen-tt 639.598 n yimezdaɣen . tezga-d deg umḍiq jar wasif azegzaw d wasif amellal .akken ad d-bdun sdat-sen asif n Nil. D tamnaḍt n lḥukuma deg Sudan anda yella umkan n uselway n tegduda, tamnaḍt n yiɣallen imenza yemgaraden, aselway n tezmert taɛessast tasudanit d tmeslayin tidelsanin n tmura tiberraniyin, amkan n kra n tiddukliyin tidelsanin taɛrabin d tafriqtin d tuget n tmettiwin tasertanin n tmurt. D ul n Tefriqt deg wayen yerzan tilelliyin n usikel, akken ad ɛeddin tilelliyin n usikel i yettqessisen ugafa n tmazirt ɣer unẓul-is akked tin i yettɛeddin deg umalu n tmazirt d usammar-is. Deg temdint-a llant aṭas n tesdawiyin, d tiɣerbazin tidelsanin d tilelliyin, timusnawen yemxalafen, tiɣimit n tkebbanin, tibankiwin tiɣelnawin d yiferrawen n tkebbanin tiberraniyin.
