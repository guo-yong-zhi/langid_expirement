Tabuzeggaɣt d aṭṭan avirusan anamsad  amanṭaṭ yettaɣ arrac imeẓyanen, igellu-asen-d kra n temrayin yettilin d timihanin di kra n tikwal. Tabuzeggaɣt d aṭṭan yettilin s tuget deg tallit n temẓi, maca yezmer ad yaɣ ula d imeqqranen. Deg useggas n 1963 ass mi id tella tadermist meqqren yiwet n tegrawt n imusnawen n ivirusen s tisselwit n iminig n Marikan John Franklin Enders tessaweṭ ad tfares tigezt mgal tabuzeggaɣt. Sin akin deg tazwara n iseggasen n 90 tigezt-a tessenger neɣ tessedres aṭṭan-a deg kra n tmura.
Aɣbalu n yigim n tbuzeggaɣt d tesfert-is d amdan. Tettenṭaṭ s unɣad  d tuṭṭsa n tɣawsiwin anda yella ubirus-a. Seld ma ad yejji umaṭun ad yeddem affal mgal tabuzeggaɣa i tanɣa .
.
Tettali tseddart n teẓɣelt n wegrud yudnen s tbuzeggaɣt i 3 wussan, anda ara ad tittaɣ tusut yekkawen d yidis d tezweɣ deg wallen-is, sin akin mi ara ad fakken 3 n wussan nni ad binent trigliwin  mellulen s agensu  n yimi-s ttemcabint ar tiqqayin n tisent. Ma deg wass wis 4 d 5 ad s-tekker tezweɣ deg uglim id ibeddun s deffir n imeẓẓaɣ sin akin ad yenfufed ar wudem d tadra w ad ikemmel ar yakk tafekka.
Isekna n waṭṭan d wid id yettbinen seld n mraw  n wussan seg unekcum n uvirus ar tfekka n umaṭun anda id igellu s tusut tagiwant d yidis amsulles  d tezweɣ deg wallen yeččarayen d imeṭṭawen yerna tellent  urnan mgal tafat akken id tezmer tawla ad tali ar 41 n tseddarin.
