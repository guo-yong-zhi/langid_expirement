Deg wass n 1 fuṛaṛ 2022, yeɣli ugrud Ṛayan Awṛam   deg yiwen n wanu yekkawen, tilqi-ines 32 m 1  di taddart n Iɣran yeṭṭafaren taɣiwant n tmurut deg ugezdu n Accawen i d-yezgan deg ugafa n Umeṛṛuk. Tamsalt-is d tkesna-ines tla-d  yiwet n txitert d umisi meqren deg umaḍal yakk, anda imelyan n yimdanen deg yakk ideggan n umaḍal zgan ṭṭafaren tamhelt n usellek-ines deg tansiwin n tmetti  d tregwa n yisalan n tiliẓri neɣ ṛadyu.
Ɣef teɣzi n 5 n wussan, llan-d warramen meqren i wakken ad ssufɣen Ṛayan seg wanu, maca ur ssawḍen ara alammi d ass n 5 fuṛaṛ. Maca ufan-t yemmut.
Di tmeddit n wass n sad 5 fuṛaṛ, telɣet-d tnarit tageldant n Umeṛṛuk ɣef tmettant n Ṛayan s usrag  n tiggas s wacu i d-yegla uɣelluy-ines s agensu n wanu send n 5 n wussan. Imir-nni ya, agellid Muḥemmed wis 6 yessawel i bab-as d yimma-as n Ṛayan i wakken ad ten-yefced .
