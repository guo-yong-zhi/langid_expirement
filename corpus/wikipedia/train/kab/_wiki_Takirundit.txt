Taqrundit neɣ Tkurundit  d yiwet n tutlayt n Bantu i ttmeslayen azal n 10.7 imelyan n yimdanen deg Burundi deg tmennaṭ n Ugaraw Ameqqranen n Tefriqt. S umata akked teglizit d tefṛansist, d tutlayin tidelsanin n Burundi, d tutlayt tayemmat n wazal n 95% n yimezdaɣ n tmurt-a. Taqrundit tettemcabi ɣer tutlayt n Kinyarwanda, d yiwet seg tutlayin tidelsanin n Rwanda. Ssin-nni n tutlayin d timsefhamt i mgaradent, yerna ttemcabin. Ddaxel n tutlayin-nniḍen, takirundit d tkinarwandait llant d yiwet n tmeslayt yettemcabin, qqaren-as rwanda-rundi, yettmeslayen-tt azal n 23 imelyan n medden.
