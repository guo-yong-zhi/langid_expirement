Urigami n tusnakt
Urigami n tusnak d timlilit n tussna d tẓuri. Ayen iceɣben inaẓuren n wurigami n tusnakt, d asemres n talɣiwin tinzegganin, amedya n umkuẓ, asrem, akerdis… S umata d timudmin timadwanin. D tamzizzelt n wanwa ara iḍefsen tamudemt yesɛan ugar n yiḍefsen ɣef tayeḍ. D asnulfu n tmudmin ideg gten iḍefsen, d tid iweɛren aṭas. Ttwalin daɣen taẓuri-a d allal i yes-s zemren ad zerwen talɣiwin n telwiḥin n ugensa amek i tezmer ad tili tigawt n tuldi deg igenni; ḍefsen-tent deg tazwara s ukaɣeḍ.
