Tawurta tazeggaɣt  d aɣersiw asuṭeṭ yeṭṭafaren tawsit n twurta deg twacult n tserɣuda yettidiren deg Marikan n ugafa, timesɣaz-is sskanayen-t-d d akken yella yettidir dina seg wazal n 1,8 n imelyan n iseggasen   
Tawurta tazeggaɣt d aɣersiw anegmar yettidir deg tẓegwa d tneẓrufin d waṭas n twennaṭin n iden
Tawurta tazeggaɣt terwes  ar telmas n iden n twurtiwin acku zdint yakk deg yiwet n tewsit, ini weglim-is imal seg uzunɣid ar waras, udem-is yeččur d acebbub, Imeẓẓuɣen-is deg-sen inẓaden d iberkanen  
Tawurta tazeggaɣt meẓẓiy acemma ɣef twurta n Kanada  maca meqqer f umcic amegrud s wazal n snat n tikkal 
Tawurta tra  ad tegmer w ad tečč igninen d iwtal maca igemmer ula d iylalen d iwuzaṭ yakk d izerzren meẓẓiyen 
Tawurta tazeggaɣt tettmercal deg taggara n tegrest ar tefsut, tadist-is tettaṭaf ugar n sin n wayyuren s kra n wussan
