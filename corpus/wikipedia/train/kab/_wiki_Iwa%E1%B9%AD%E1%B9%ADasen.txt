Iweṭṭasen  d yiwet n tewsit  tamaziɣt yeṭṭafaren afurk n yiznasen, nebḍen  Ameṛṛuk deg tallit ger n 1472 d 1541. Iwaṭṭasen ulin ɣer udabu deffir n uɣelluy n Wayt Mrin. D acu kan, ass mi inebḍen Yiwaṭṭasen timura tinselmin n Tefriqt n Ugafa, tufa-tent-id tegnit ɣlint yakki di tallit n urakam , yerna Ispenyuliyen d Yipuṛtugaliyen llan ẓeddmen-d ɣef teftisin n Tefriqt n Ugafa akken ad ṭṭfen timdinin i d-yezgan ɣef yiri n yilel . Tallit n Yiwaṭṭasen d tin i d-yusan deg wakud n uɣelluy n tgelda n Ɣernaṭa deg useggas n 1492 ayen i d-iglan s ufukku n udabu n yinselmen deg Wandalus.
Ddaw n udabu n Yiwaṭṭasen, yebded  unnerni n temdinin n Umeṛṛuk, yerna simal tella tettnerni tezmert n Yespenyuliyen d Yipuṛtugaliyen deg Tefriqt n Ugafa, ayen yeǧǧan tawacult n Yiseɛdiyen ad kkren deg useggas n 1541 w ad sseɣlin awanak n Yiwaṭṭasen.
