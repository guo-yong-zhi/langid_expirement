Taserẓut uzumba neɣ Aṭeggal azegzaw  d talmest n igṭaṭ yeṭṭafaren tfesna n ijqiwen deg twacult n tmeserẓatin yakk d tewsit n temserẓa, Učči-ynes yezzi af iɣlifdaren deg unebdu yakk d yifsan deg tegrest. Tasemhuyt n usegdef n telmest-a tettili-d gar mayu d yunyu. Ibennu agdef-is ɣef wuruzen n isekla , deg yal tikkelt n usegdef tawtemt tessrusay gar n Kṛad ar kuẓ  n tmellalin. 
Taserẓut uzumba d talmest tamyiwent n yigṭaṭ  amuɣel n Tmurt Taqbaylit, anda ur tesegdud kan  kra n tiẓegwa n taydiwin n ugafa n tmurt. Isem-is ussnan yettak aslulli  i Jean-Paul Ledant amussanw amaswaḍ abiljik i d-yesnarmen talmest-a deg wayyur tuber 1975, aglam n ufrux yetwexdem sɣur amesnaylal Afrensaw Jacques Vielliard. Isali n tagrut agi yessewhem aṭas amadal n tasnaylaly daɣen isali agi yexdem tuzzya n umaḍal. Taserzut uzumba tella tudes  ar Sitta n Krüper . yegra-yas yiwen kan azwu n uzuzer , yettwaggezen sɣur times, arad d tigawt talsant ; talmest tettwali dɣa am « deg umihi » sɣur Tadukli tagraɣlant i tagaẓt n ugama.
