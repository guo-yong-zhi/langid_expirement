Muhammad al-Hajj Salam Amziane ilul deg taddart n At Buxlef deg tewsit n At Weriaɣel deg tagara n useggas n 1925 Yeɣra lquran deg tmesgida n taddart akked Jabala. Ikemmel leqraya-ines deg tazwara n yiseggasen n 40 n lqern wis 20 deg tesdawit n Al-Asil  deg temdint n Al-Hoceima, dɣa yerra abrid ɣer temdint n Fes akken ad ikemmel leqraya-ines deg tmesgida n Al-Qarawiyyin, anda i d-yewwi asebter amatu. Deg tazwara n yiseggasen n 50, ixeddem d aselmad deg yiwet n uɣerbaz deg taddart n Bamhamed, ɣer tama n Fas, yerna yeqqim deg taddart-a ugar n rebεa iseggasen. Dɣa yeqqim kra n lweqt deg Tetouan.
