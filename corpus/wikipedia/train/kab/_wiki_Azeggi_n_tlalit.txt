Azeggi n tlalit neɣ Iḍ n tlalit d yiwen n yiḍ iy zeggarayen ass n talalit, itteɣlay-d deg wass n 24 dujamber ɣer yilan ikritiyen utrimen, yakk d wass n 6 dujamber ɣer yilan ikritiyen isemranen.
Isem n yiḍ n tlalit yettuɣal ɣer wawal n yiḍ, acku di teswast tudayt n zik yella ibeddu degs wass seld n uɣelluy n tfukt, ihi d anecta yeǧǧan talalit ad tebdu deg deg yiḍ yezwaren ass n tlalit.
Yettili-d usfugel s uzeggi n tlalit deg waṭas n tmura d yideggan n umaḍal, anda i yeqqen s yimensi n tlalit d tmudemt n Baba Nwal yettwassnen s tenfusin-is d tufka-ines i warazen d yisefkan i warrac meẓẓiyen.
