Abucenbel amaziɣ neɣ aɣerda aṛumi neɣ Abadaw  d yiwet n telmest n temseɣẓaẓ yeṭṭafaren tawsit n ubucenbel deg twacult taɣeṛdayt, Talmest n uɣeṛda ayi tella tettidir s waṭas deg irawen n temdinin n Lezzayer d Umeṛṛuk, maca ar tura zeddiɣ  yettidir s tuget deg tuddar d igran n tmura ayi gar-asent Tamurt n Iqbayliyen, Abucenbel ur iga ara d asertay  s unemgal n uɣeṛda wexxam acku yettanef tudsa  ar ufgan
Abucenbel meqqer cwi ɣef tezɣuda, Asallaf-is yugar cwi aqerruy-is d tafekka-ynes
Yettidir deg Tefriqt n ugafa, Deg yideggan inefsiren  n Umeṛṛuk d Lezzayer d Tmurt n Iqbayliyen d Tunes yakk d temdint n Sebta d Mlilt yeṭṭafaren tamurt n Spanya
