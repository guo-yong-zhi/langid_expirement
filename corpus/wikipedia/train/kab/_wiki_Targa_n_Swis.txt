Targa n Swis  d targa n waman tamgurant tusligt n tukkit, tezga-d deg ugafa asemran n Miẓra, tessawaḍ gar n yilel agrakal d yilel azeggaɣ s teɣzi n 193 km. S uzegrar tebṭa ɣef sin n imuren deg ugafa d unẓul n igelmimen irẓaganen, ma di tehri tebḍa ɣer sin isuka deg wamur meqqren n yimuren-is i wakken ad teǧǧ iɣerruba ad zegren ɣef snat n tniliwin deg yiwen n salud gar n Uṛuppa d Asya. Dɣa tettumeggez d asaku awlal arurad yakk gar n sin n yimenẓawen-a yerna tessegfar-d azal n 15 n wussan seg wakud n ussikel seg yixef n usirem agerram.
Tikti n usnulfu n terga-ayi tebda-d deg useggas n 1798 ass mi i d-yebda usizzel n Fṛansa ɣef Miẓra, sin akin yekta  Napoléon Bonaparte ad iɣez targa, maca asurif-a ur yessaweḍ ara ad yili. Deg useggas n 1854, yessaweḍ Ferdinand de Lesseps ad yessenẓ  Muḥammad Saɛid s usenfar-a
