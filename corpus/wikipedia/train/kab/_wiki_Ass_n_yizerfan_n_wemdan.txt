Ass n yizerfan n wemdan d talemmiẓt tamseggast s wacu yesfuggul yakk umadal yal ass n 10 dujamber. Yettwafran wass-a sɣur n ugraw amatu n yeɣlanen yeddukklen deg wass n 10 dujamber 1948, ass mi yegguni ɣef ulɣu agraɣlan i yizerfan n wemdan. 
Dɣa d alɣu amadlan ayi i d amezwar deg tɣult n yizerfan n wemdan akken iga d yiwen seg imsedṛuyen imeqranen n tuddsa n Yeɣlanen Yeddukklen. Yella-d usbeddi n wass n yizerfan n wemdan s talɣa tamaddudt deg umsegraw uṭṭun 317 n ugraw amatu deg wass n 4 dujamber 1950 ass mi i d-ilɣet  ɣef tneɣtust uṭṭun 423, s usnebget n yakk igmamen n tmura d tuddsiwin yettaken assaɣ i wesfuggel s wass-nni.  1
