Bisan , d yiwet si temdinin tiqburin n Falesṭin, tezga ass-a deg twilayt n ugafa n Yisrayil ɣef 83 km deg ugafa-utram n Lquds. Tamdint-a tesεa azal n 7,330 dunm , deg-s 16,200 n yimezdaɣen, tuget deg-sen d Udayen deffir n tuffɣa n Waɛraben n Falisṭin deg ṭṭraḍ n 1948. Tamdint n Besan tezga deg wul n Marj n Besan i d-yettawḍen ɣer Ugaraw n Urdun s Marj n Ibn Amer, deg unẓul-ast n temdint n Naṣira, deg usammer n temdint n Jenin n Falesṭin, deg unẓul n temdint n Um Qays n Urdun d wasif n Urdun. Tamdint-a tella tettwaḥseb d tamnaḍt n Bisan asmi i d-teɣli n Nakba d usnulfu n Israyil deg useggas n 1948.
