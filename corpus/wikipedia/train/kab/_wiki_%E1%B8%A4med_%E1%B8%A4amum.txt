Tameddurt
Ahmed HAMOUM, d yiwen n wemdan amagnu yettidiren am ṭaqqa n yimesdurar di tmurt n Yeqbayliyen. Ilul ass n 04 di meɣres 1955 di taddart n Taqerbust, taɣiwant n Aɣbalu, tamazzya n Imceddalen, agezdu n Tubiret.    Almud-is amezwaru, yesɛedda-t deg uɣerbaz n taddart-is. Imi ur as-tefki ara twenza-s ad yekcem aɣerbaz alemmas am tezyiwin-is, ɣas akken yewwi-d aselkin n tezrawt amezwaru , ikemmel deg yiɣerbazen usligen .
    Yeǧǧa-n igen, yeqqel d aɣarim, yesselmed deg uɣerbaz amezwaru s teɣzi n 18 n yiseggasen. Segmi yekki di tannant n yiselmaden n tmaziɣt i d-tsuddes tuqqna taɣelnawt  n umussu adelsan amaziɣ  di Tidukla Tadelsant Tamaziɣt n Bgayet timsirin n tmeddit, si 23 Duǧember 1995 armi d aggur n yulyu 96, ibeddel ɣer Tmaziɣt deg wunbir 97 armi d-yeffeɣ tastaɣt  di taggara n duǧember 2010.
