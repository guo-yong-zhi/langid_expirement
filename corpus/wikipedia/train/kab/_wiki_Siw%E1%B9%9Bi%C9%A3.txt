Sawṛaɣ neɣ Tajɣuɣt d kellu  n weglim d imelli n tiṭṭ s yini awṛaɣ d uzegza s usrag  n walluy deg uswir n ubilirubin. Deg waṭas n tegnatin igellu-d s tuččit, Siwṛiɣ d aṭṭan id yettmagen i wugar n wezgen n warrac deg imalas  amezwar seld tlallit maca  tuget n tegnatin ur ttilint ara d timihanin. Ma yella iswiren n Bilirubin ar warrac meẓẓiyen ulin s waṭas yerna yeṭṭef akud ɣezzifen yezmer ad iglu s telmest n uttwaɣ n wallaɣ yettwassnen s siwṛiɣ aɣisan. 1  2  3  4  5 
Isragen n siwṛiɣ d wid igan gar n isragen armihanen d yisragen iteɣɣen . 
Deg tuget n tikwal,aswir n Bilirubin deg idammen yettili s ddaw n 1mg/dl. Iswiren i yuraren 2 ar 3mg/dl gellun-d s siwṛiɣ, Dɣa Abilirubin yebṭa ar sin n inawen: Abilirubin usrid d ubilirubin arusrid
