Kabul d tamaneɣt n Afɣanistan, tezga-d deg ugafa-agmuḍan  n tmurt. Zedɣen-tt 2.850.000 n yimezdaɣen. Deg useggas n 1774 ay teqqel temdint-a d tamaneɣt n tmurt. Ass-a, Kabul d nettat ay d ammas n tdamsa, n tnezzut  akked yedles n Afɣanistan, yerna ɛlayet 1800 n lmitrat. Aṭas n yimenɣiyen ed yemgura  ay yekkan ɣef Kabul deg lqern wis 19 ed wis 20. Deg yiseggasen n 1980, asmi ay tt-ṭṭfen Yisuvyatiyen, akked yiseggasen n 1990, asmi ay yenker wemgaru gar Wafɣanen, aṭas n yimukan ay ihudden seg temdint-a.
