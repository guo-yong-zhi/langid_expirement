Wuhan  d tamenrut n iɣir n Hubei deg tlemmast n tmurt n Ccinwa, Wuhan d tamdint ilan  akk aṭas n imezdaɣ deg Hubei. Akken i tga d tamdint tameqrant tis 7 di Ccinwa, Yezga-d deg usamer n Ccinwa akken i tga d ammas amugran d udamsan ilan  azal meqqren deg tlemmast n Ccinwa
Deg ussan ineggura n yennayer 2020 tamdint n Wuhan tettwamdel yerna tettusinef s usrag  n usefruret d unfufed n waṭṭan n Covid-19 id yekkan seg ulzuz n Wuhan i yimeččiyen iwlalen seg imaren medlen ula d alzuz-nni
Yettwasnirem-d waṭṭan n Covid-19 i tikkelt tamezwarut deg temdint n Wuhan id yezgan di tlemmast n Ccinwa deg wayyur n dujamber 2019. Imusnawen ttɣilen d akken aṭṭan-a yekka-d seg iɣersiwen sin akin ineṭṭ-d ar yimdanen s iɣersiwen-nni ig yuɣ waṭṭan-a deg ulzuz n tnezzut n iɣersiwen, sin akin yerna Yenfufed ar igezda n iden n Ccinwa deg wussan imezwura d ilemmasen n wayyur n yennayer s usrag n tfaskiwin n yixef useggas n Ccinwa
