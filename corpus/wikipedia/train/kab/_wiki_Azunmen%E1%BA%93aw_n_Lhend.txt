Azunmenẓaw n Lhend ,  d amur n wakal meqqren deg umenẓaw n Asya tettawi timura yellan deg Tafilt tatektunt n Lhend am: Bangladec, Pakistan, Lhend, Butan, Nipal akk d tegzirin id yezzin am Sri Lanka d Maldiv.
Azunmenzaw n Lhend d akal id yezgan si tama n wadda n idurar n Himalaya akk d agni n Tibit, Deg tama n uneẓẓul yezzi-yas-d ugaraw amelwi, Seg umalu ilel aɛrab.
Azal n 1,700,000,000 n ifganen ttidiren deg azunmenẓaw n Lhend, gar-asen ugar n 65% ttidiren deg Lhend, Ttmeslayen aṭas n tutlayin .
