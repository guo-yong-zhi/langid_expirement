Arram amesguceṛ unɣid  d talmest gar n inawen n unagi ussnan anarum aladɣa deg tɣult n tesnujjya. Anda imdanen ɣef wacu id tettili tezrawt ttwasennefrun s ugucar  s yiwet neɣ ugar n tarrayin n ussuji yemgaraden yettilin deg tezrawt. Dɣa irramen imesgucaṛen unɣiden ttumeggzen d akayad n umnir deg warramen ilektuyen yerna yettwasemres  i wkayad n tmellilt n inawen yemgaraden n inekcumen isnajjyanen. Akken id yezmer ad yefk isalan ɣef tasedmirt tansafart tuɣnift.
Irmawen "RTC" d Tirmit tagacuṛant ttwasemrasent deg kra n tikwal am igdawalen, maca tallilt tamalxayt tanarrayt tga d taggaẓt n RTC s isem kan i warramen ilan  tigrawin n usemyif
