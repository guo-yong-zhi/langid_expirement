Skutlanda , d tamurt iḍefṛen i Tagelda Yedduklen, deg ugafa n Turuft. Tamanaɣt-is d Idinburg.
Tageldit n Skutland, di lqern wis 9, deg useggas n 1707 tesbedd tageldit n Briṭanya Tameqqrant akked Legliz deg umtawa asertan gar-as d Legliz. Deg tdukli-ya, Skutland ass-a, akked Legliz,d Wilz, akked Irland n Ugafa, sbeddien d Tagelda Tameqqrant n Bretanya .
Deg Skotland llant aṭas n temdinin yesɛan ugar n 100,000 n yimezdaɣen , gar-asent Glasgow s 654,542 n yimezdaɣen, d Edinburgh tamaneɣt s 421,213 n yimezdaɣen, d timdinin n Aberdeen d Dundee. 
Skutland tesεa tuffɣa n yimezdaɣ-is, s ugar n alef n yimdanen  ttaǧǧan-tt mkul aseggas s sebba n uxuṣṣ n uxeddim.
Llant Tlata tutlayin di Skutland d taglizit, tiscottit d tgaylit. 
Aṭas n yimezdaɣ i ttmeslayen taglizit tiskuttit Deg tegzirin n Skotland, imzdaɣen ttmeslayen tutlayt tagylit, nettat d tutlayt taqburt.
