Tamdint n Napuli ) d tamdint tis kraḍ i imeqqren akk deg Ṭṭelyan, deffir Roma ed Milano, akked ahat n umelyun n yimezdaɣen 1 , tezga-d deg ugemmaḍ umalu n Ṭṭelyan, deg unẓul-usamar n Roma  2 . Napuli d tamanaɣt n tamnaḍt n Kampanya / Campania 3  d tamnaḍt tadelsant n unẓul n Ṭṭelyan, gar 3 d 4.4 imelyan n medden i izedɣen deg temnaḍt-a. Deg tazwara, tamdint-a tella d taɣiwant tagrigit isem-is Neopolis dɣa teɣli-d sdat n tefriqt tarumant. Seg taggara n tallit tan tlemmast ɣer lqern wis 18 n leεmer n Lmasiḥ, tamdint-a tella tettwaḥseb d yiwet seg temdinin timeqranin akk akk deg Tuṛuft. Amezruy n tsertit n Naples yettwabna seg wasmi i tt-frnen d tamaneɣt n tgelda n umalu n Ṭṭelyan deg tallitin yemgaraden. Aṭas n yimezdaɣ n Naples i yettmeslayen taṭelyanit s yiwet n tmeslayt yemgaraden s waṭas ɣef taṭelyanit tatrart.
Napuli seg awal talatint : Neapolis, anamek-is d tamdint tamaynut 2 .
