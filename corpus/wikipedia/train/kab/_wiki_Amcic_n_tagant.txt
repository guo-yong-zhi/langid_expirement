Amcic n tagant  d aɣersiw amseksum yeṭṭafaren tawacult n tserɣuda, aɣersiw-a yettusismel deg tserɣuda timeẓyanin maca d netta yakk i d ameqran deg tewsit n yimcac 
Teɣzi n uɣersiw-a d tin yettawaṭen deg tlemmast ar 70 cm, asallaf-is meẓẓiy ur yugar ara 20 cm, Addud-is ar tuyat ila azal n 36 cm, azuk-is gar n 4 ar 16 kg, ini-ynes yemgarad s unect  n telmest n tedfert seg uzunɣid uwriɣ ar waras uzwiɣ, Akken id teffaɣen izirigen deg uzagur n wammucen maca tekksen mi ara ad ddun deg waddar  
Awurta n ubelma d isem wis sin n uɣersiw-a ɣas akken ulac assaɣ gar-as d gar n twurtiwin anagar amcabi deg wanẓaden yellan deg yixef n imeẓẓuɣen-is
Amcic n tagant d aɣersiw yettidiren deg Aneẓul n Asya  d ccam d yirawen n wasif n nnil
Talmest-a tettugred  yakk d wemcic n wexxam ayen id yeglan s telmest ur tettagad ara aman isem-is chausie
