Deg tesnudert, Amenkud  d yiwet n taggayt tamessartit id yezgan sufell n tgelda, tikwal tettwameggez  d akken-it d anamek n irem n tɣult igan d atrae fell-as. Ɣas akken sin n yirmanwen-a ur zdin ara yakk, s umaẓlay irem n umenkud d win yifen taɣult deg usatal n tinnagrawt tanemhazt 
S umazray, d irem n umenkud id yettussumren d amezwaru. S terwest d uɣmar  n yiwen n umenkud ɣef tgeldiwin yettwaɣen
