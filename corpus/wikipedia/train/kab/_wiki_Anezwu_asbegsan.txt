Anezwu asbegsan  d anezwu i d-yezgan deg tlemmast n tegnit gar n sin ijeṛdan n tehri 10° n ugafa d uneẓẓul n tiṭṭ n tfukt, yettwasen s weẓɣal d walluy n tseddaṛin ɣef teɣzi n useggas, d tuget n ugeffur ayen yettaǧǧan  tiẓegwa tisbegsatin ad ugtent deg yideggan-nni, Tiẓgi n Amaẓun d tin yettwassnen yakk gar-asent.
Yettili deg ugafa n Tamrikt n Wenẓul, timura n Aneẓul n usamar n Asya, anẓul n Sudan, Tafriqt talemmast d Ɣinya tamaynut.
