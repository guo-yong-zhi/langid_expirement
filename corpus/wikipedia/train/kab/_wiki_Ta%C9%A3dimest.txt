Taɣdimest neɣ Tamaziɣt n Ɣdames  d yiwet n tutlayt tamaziɣ tasemrant yeṭṭafaren afurk n tutlayin tiznasin tisemranin. Tutlayt-a ttmeslayen-tt yimezdaɣ n yimedwen n Ɣdames deg utaram n Libya, di tlisa-ines yakk Lezzayer d Tunes. 
S unect n tensa n Ethnologue, llan wazal n 10,000 alamma d 13,000 n yimdanen yettmeslayen s tutlayt taɣdimest deg useggas n 2006, akken i llan wazal n 2000 n yimdanen niḍen yettmeslayen yis deg beṛṛa n tama n Ɣdames. Dɣa tutlayt-a d tin yettwaggzen s umihi n wengar s tseddaṛt n 6b.
Taɣdimest d yiwet n tutlayt tamaziɣt iy ggẓen  aṭas n tulmisin tisnilsanin tisufin, dɣa amawal n teɣdimest yettwaskelsen sɣur n Lanfry immal-d asemdu idrusen seg tutlayt taserɣint .
