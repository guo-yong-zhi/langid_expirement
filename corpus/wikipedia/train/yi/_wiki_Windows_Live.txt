Windows Live  איז דער קאלעקטיװער מארקע־נאמען פאר עטלעכע באדינונגען און קאמפיוטער פראגראמען פראדוקטן פון מײקראָסאָפֿט. די מערהײט פון די באדינונגען זײנען װעב־אפליקאציעס , אבער עס זײנען דא אױך אפליקאציעס װעלכע מען דארפ אינסטאלירן. עס זײַנען פֿאַראַן דרײַ װעגן מיט װי אזױ Windows Liveס באַדינונגען זײנען אָנגעבאָטן: אַלגעמײנע װינדאָס־לײװ אַפליקאציעס, װעב באַדינונגען און מאביל באדינונגען.
װינדאָס־לײַװ איז צום ערשטן באַפֿרײט געװאָרן אין נאוועמבער 1, 2005. בײ איר ערשטן באַפרײונג האָט װינדאס־לײװ ארײנגערעכנט א טײל אײגנשאַפֿטן פֿון דער MSN, אבער פֿונדעסטװעגן איז MSN געבליבן זײענדיק צו דער זײט פון װינדאס־לײװ.
די װינדאָס־לײַװ װערט נענטער און נענטער צו דער Windows אפערירן סיסטעם, למשל אין Windows Vista איז אײַנגעבױט א לינק אראפצולאדירן דעם Windows Live Messenger און אין Windows 7 װעלן די װינדאָס־לײװ אַפליקאציעס פֿאַרבײטן די Windows Mail, Windows Movie Maker און די Windows Photo Gallery.
א טײל פֿון די װינדאָס־לײַװ באַדינונגען זײַנען: Live Search, Windows Live Messenger, Windows Live Hotmail, Windows Live OneCare, און די Windows Live Spaces.
