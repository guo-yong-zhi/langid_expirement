Ang MS-DOS  ay isang operating system na ginawa ng Microsoft para sa komersyo.  Ito ang pinaka-karaniwang ginagamit na kasapi sa pamilya ng operating system na DOS at ang namamayaning operating system para sa platapormang PC compatible noong dekada 1980.  Unti-unting napalitan ito ng iba't ibang henerasyon ng operating system na Windows para sa mga kompyuter na pang-desktop. Ngunit, ito parin ay isang applet ng mga bersyon ng Windows mula sa Windows 2000 at ang applet file nito ay cmd.exe. Ito ay nahahanap sa /%SystemRoot%/%Windows Folder%/System32.
 Ang lathalaing ito  ay isang usbong. Makatutulong ka sa Wikipedia sa pagpapalawig nito.
